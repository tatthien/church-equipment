# Church Equipment Management

Webapp quản lý thiết bị nhà thờ.

## Tech Stack

- **Monorepo**: npm workspaces
- **API**: Bun + Express + SQLite
- **Frontend**: Next.js + Mantine

## Getting Started

```bash
# Install dependencies
bun install

# Run development servers
bun run dev
```

- API: http://localhost:3001
- Frontend: http://localhost:3000

## Project Structure

```
packages/
├── api/     # Backend API
└── web/     # Frontend
```
